Document Objetc Model

Conjunto de Objetos dentro do seu navegador que vai dar acesso aos componentes internos do seu website

Árvore DOM

Toda árvore começa com a Window. É um objeto DOM

Location: URL, páginas atual e anteriores

Document: documento atual. Dentro dele existe o HTML. E dentro dele, existem mais dois filhos chamados head e body.

Existem as nomenclaturas parent e child

History: guarda todo o caminho percorrido

Métodos de seleçao dos Elementos da Árvore DOM

  • Marca

    • Método: getElementsByTagName()
    Dessa forma, é possível selecionar mais de um objeto, vez que existem vários objetos do mesmo tipo. A tag de parágrafo é um exemplo.

    Este é o 11º parágrafo, que será usado de exemplo para a seleção via Tag.

  • ID

    • Método getElementById()
    Aqui já é possível selecionar as tags que possuem identificação pessoal

    Este parágrafo possui a id "testeID".

  • Nome

    • Método: getElementsbyName()
    Seleção de elementos que possuem nome.

    Este parágrafo possui o name "testeName".

  • Classe

    • Método: getElementsByClassName()
    Seleção de elementos que possuem classe determinada

    Este parágrafo possui a class "testeClasse"

  • Seletor

    • Método: querySelector() e querySelectorAll()
    Seleção de elementos HTML através de class (.) ou ID (#) como parâmetro do método entre áspas simples ou duplas.

    Este parágrafo possui a class "testeSeletor".

    Este parágrafo possui o ID "testeSeletorID".
